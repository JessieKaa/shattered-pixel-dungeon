# Remixed Dungeon sprite asset map

Source: `../remixed-dungeon/sprites/` (relative to this worktree)
Destination: `core/src/main/assets/mods/remixed_full/sprites/`

This file maps every original remixed sprite filename to its imported location
inside the `remixed_full` builtin mod. It is intended as a reference for the
script-rewrite milestone that will make `LuaItem`/`LuaMob`/etc. reference these
assets at runtime.
